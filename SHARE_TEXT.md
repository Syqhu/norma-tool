# 共有用文章

## HoYoLAB / SNS向け

```text
ZZZ用に「norma tool」という育成管理アプリを作りました。
今はβ版です。

キャラの育成方針を見たり、目標ステータスと比べたり、ディスクをスコア化したり、編成を考えたりできます。
HoYoLAB同期、スクショ読み取り、所持ディスク倉庫、ビルド完成度チェック、GitHubのアップデート確認も入れています。

まだ調整中の部分もありますが、使いながら少しずつ直していく予定です。

追加予定:
・OCR読み取り結果の確認/修正画面
・ディスク比較機能
・売却/餌候補リスト
・危局/式輿向けのA面B面編成管理
・HoYoLAB同期結果のレビュー画面
・キャラ別の編成相性データ追加
・HoYoLAB連携データを使ったユーザー間ランキング

GitHub:
https://github.com/syqhu/norma-tool

個人制作の非公式ツールです。
未署名アプリなので、初回起動時に Windows の警告が出ることがあります。
```

## GitHub Release向け

```md
## norma tool v0.1.1 beta

ZZZ向け育成管理アプリ「norma tool」のβ版です。

### 今回の追加

- GitHub Releasesのアップデート確認
- 起動時の自動チェック設定
- 設定画面から最新リリースを開くボタン

### 入っているもの

- 育成方針、目標ステータス比較
- スクショから現在ステータス読み取り
- ディスク入力、スコア計算、交換候補表示
- 所持ディスク倉庫
- キャラ別スコア基準
- 編成シミュレーター
- キャラ相性を見た編成評価
- ビルド完成度バー
- HoYoLAB同期
- 育成優先度ダッシュボード
- デイリー未完了通知
- GitHub Releasesのアップデート確認

### 使い方

`norma-tool-win-unpacked.zip` を展開して、`norma tool.exe` を起動してください。

### 注意

個人制作の非公式ツールです。  
未署名アプリなので、初回起動時に Windows の警告が出ることがあります。
```
