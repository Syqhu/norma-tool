# norma tool

ZZZ向けの育成管理デスクトップアプリです。  
まだβ版なので、使いながら少しずつ直していく予定です。

キャラの育成方針を見たり、目標ステータスと比べたり、ディスクをスコア化したり、編成を考えたりできます。  
HoYoLAB同期、スクショ読み取り、所持ディスク倉庫、ビルド完成度チェックにも対応しています。

## Download

Release から `norma-tool-win-unpacked.zip` をダウンロードして展開し、`norma tool.exe` を起動してください。

未署名アプリなので、初回起動時に Windows Defender / SmartScreen の警告が出ることがあります。

## Features

- キャラ一覧とアイコン表示
- キャラ別の育成方針
- 目標ステータスと現在ステータスの比較
- スクリーンショットから現在ステータスを読み取り
- ディスク入力、スコア計算、交換候補表示
- 5番/6番ディスクのメイン効果OCR補正
- 所持ディスク倉庫
- キャラ別スコア基準
- 編成シミュレーター
- キャラ相性を見た編成評価
- ビルド完成度バー
- HoYoLAB連携による所持キャラ/ステータス/ディスク同期
- HoYoLAB同期差分表示
- 育成優先度ダッシュボード
- デイリー未完了通知
- GitHub Releasesのアップデート確認

## Data Policy

目標ステータスは確認できたものだけ登録しています。  
未確認キャラには適当な数値を入れず、未確認として扱います。

参照元はアプリ内の「ソース」タブにまとめています。

HoYoLAB連携はアプリ内の専用ログイン画面を使います。Cookieやログイン情報を外部サーバーへ送信する機能はありません。
