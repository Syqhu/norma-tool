## norma tool v0.1.2 beta

ZZZ向け育成管理アプリ「norma tool」のβ版です。

### 今回の追加

- ディスク画像OCR結果を倉庫へ自動保存
- HoYoLAB同期ディスクを倉庫へ自動保存
- 現在装備と比較候補のディスク差分表示
- 編成画面の相性候補表示
- 育成素材プランナーの日数/優先順表示
- キャラ別タグ管理
- 初回セットアップチェックリスト
- 保存データ破損時に起動が落ちにくくなる修正

### 入っているもの

- 育成方針、目標ステータス比較
- スクショから現在ステータス読み取り
- ディスク入力、スコア計算、交換候補表示
- 5番/6番ディスクのメイン効果OCR補正
- 所持ディスク倉庫
- キャラ別スコア基準
- 編成シミュレーター
- キャラ相性を見た編成評価
- ビルド完成度バー
- HoYoLAB同期
- 育成優先度ダッシュボード
- デイリー未完了通知
- GitHub Releasesのアップデート確認
- ディスク倉庫へのOCR/HoYoLAB自動保存
- キャラタグ、比較候補、初回セットアップ

### 追加予定

- OCR読み取り結果の確認/修正画面
- ディスク比較
- 売却/餌候補リスト
- 危局/式輿向けA面B面編成管理
- HoYoLAB同期レビュー
- 編成相性データ追加
- HoYoLAB連携データを使ったランキング

### 使い方

`norma-tool-win-unpacked.zip` を展開して、`norma tool.exe` を起動してください。

### 注意

個人制作の非公式ツールです。  
未署名アプリなので、初回起動時に Windows Defender / SmartScreen の警告が出ることがあります。
